Build a modern frontend UI for an AI Customer Support SaaS application called **SupportMind AI**.

The app should look like a professional SaaS product where businesses can create an AI chatbot for their website by uploading company documents, FAQs, and support content.

Build **frontend only**. Use dummy/static data where needed. Do not create backend logic, database logic, APIs, authentication backend, AI logic, or server-side functionality.

Tech stack:

- Next.js
- TypeScript
- Tailwind CSS
- shadcn/ui
- Framer Motion
- Lucide React icons

Design style:

- Modern SaaS dashboard
- Clean dark/light UI
- Soft gradients
- Glassmorphism cards
- Rounded corners
- Professional spacing
- Responsive design for desktop, tablet, and mobile
- Smooth hover effects and page transitions
- Clean typography

Pages to create:

   Create a professional SaaS landing page with:

- Navbar with logo, links, login button, and “Get Started” button
- Hero section with headline: “Launch an AI Support Chatbot Trained on Your Business Knowledge”
- Subtitle explaining that businesses can upload FAQs and documents to create a customer support chatbot
- CTA buttons: “Start Free” and “View Demo”
- Hero dashboard preview mockup
- Feature section
- How it works section
- Pricing section
- Testimonials section
- FAQ section
- Footer
